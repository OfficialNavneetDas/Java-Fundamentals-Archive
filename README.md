# Java Fundamentals Archive

This repository is a personal code archive containing early programming exercises and projects written in Java. It serves as a backup of foundational coursework and logic building exercises before transitioning focus to Python and AI development.

## 📂 Repository Structure

The code is organized into four main categories:

### 1. Core Basics (`/01_Core_Basics`)
Basic syntax, standard I/O operations, and fundamental control flow.
* `Addition`, `String_input`, `import_from_user`
* `Conditions`, `Looping`

### 2. Object-Oriented Concepts (`/02_Object_Oriented`)
Early implementations of Java's OOP principles.
* `Functions`
* `Inheritance`

### 3. Algorithms & Patterns (`/03_Algorithms_and_Patterns`)
Logic-building exercises and mathematical problem solving.
* **Math:** `Prime`, `Factorial`, `Fibonacci_Series`, `Armstrong`, `Palindrome`
* **Visual Logic:** `Pattern`, `Advance_Pattern`
* **Utilities:** `Number_name`

### 4. GUI & Graphics (`/04_GUI_Graphics`)
Introductory graphical user interfaces and event handling.
* **Components:** `learn_buttons`, `text_field`, `form`, `Front_page`
* **Event Listeners:** `learn_actionListener`, `learn_actionListener2`
* **Layouts & Painting:** `layoutManager`, `ndpaint`, `web_page`
* **Mini-Project:** `Making_calc`
